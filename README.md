# libil2cpp Patcher
 ![](https://i.imgur.com/NvI3fEL.png)
 
It's just a simple operation.
1. Drag and drop .so file into libil2cpp Patcher.
2. Input Offset 
3. Input Bytes (Hexadecimal)
4. Press patch

Also support load / save with xml.
